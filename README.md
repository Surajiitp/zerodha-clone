# Zerodha Clone

A full-stack Zerodha Clone built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

- User Dashboard
- Holdings Management
- Positions Tracking
- Orders Section
- Funds Management
- Responsive UI
- MongoDB Integration

## Tech Stack

### Frontend
- React.js
- Bootstrap
- Material UI

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

## Project Structure

```text
zerodha-clone/
├── backend/
├── dashboard/
├── fronted/
```

## Installation

### Clone Repository

```bash
git clone https://github.com/Surajiitp/zerodha-clone.git
```

### Frontend

```bash
cd fronted
npm install
npm start
```

### Dashboard

```bash
cd dashboard
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm start
```
